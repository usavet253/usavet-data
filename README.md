# usavet-data

Public data feed for USAVET.AI Daily Accountability Index. Provides structured JSON for real-time website updates across veteran and military family indicators.

## USAVET.AI Data Feed

This repository hosts the public data files used to update the USAVET.AI platform.

## Files

- `public_daily.json` → Primary public data source for website and future API consumption
- `history.json` → Historical data used for trend analysis and longitudinal display

## Purpose

Provides approved public-facing JSON outputs that power live updates across the USAVET.AI platform.

## Notes

- Data is updated regularly
- This repository supports front-end and public delivery only
- Scoring methodology is not disclosed
- Official score generation occurs in the private backend repository
